# Wikipedia Bingo

A real-time multiplayer Chrome extension game where players navigate Wikipedia to complete dynamically generated bingo boards.

## Features

- Chrome extension client
- Backend API (Node.js + TypeScript)
- PostgreSQL persistence
- Real-time multiplayer (planned)
- Lockout bingo mode (planned)
- Kubernetes deployment (planned)
